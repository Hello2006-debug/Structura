# Structura

> Making Data Structures Beautiful — and Critical.

An interactive web application for learning data structures through 
scroll-driven storytelling, live animation, and a critical historical lens.

## Built With
- Next.js 14
- Tailwind CSS v4
- Anime.js
- Python / FastAPI (coming soon)
- PostgreSQL (coming soon)

## Data Structures Covered
- 01 Linked List
- 02 Stack
- 03 Queue
- 04 Tree (BST)
- 05 Graph
